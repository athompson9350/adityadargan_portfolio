## 📋 How to Run This Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/Suraj-G-Rao/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Install dependencies**
   ```bash
   yarn install
   # or
   npm install
   ```

3. **Start the development server**
   ```bash
   yarn dev
   # or
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
